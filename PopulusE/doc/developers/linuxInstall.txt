== Install Linux JRE ==

Download JRE into /usr/java/
#mkdir /usr/java
Download from http://www.java.com/en/download/manual.jsp
* The file name should be like jre-1_5_0_06-linux-i586.bin

Change the permission of the file you downloaded to be executable. Type:
# chmod a+x jre-1_5_0-linux-i586.bin

Start the installation process.Type:
# ./jre-1_5_0-linux-i586.bin

Add java to the path
# export PATH=$PATH:/usr/java/jre1.5.0_06/bin/
(Add to your startup file(s) to make this permanent.)

Verify JRE installation
# java -version
(Should give the version)

== Install Populus ==

Save the Populus bin file.

Make it executable
# chmod +x PopInstall.5.4.06Jan06b.bin

Install
# ./PopInstall.5.4.06Jan06b.bin

After install, Populus is in menu and /usr/local/bin/
